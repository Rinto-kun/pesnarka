# 656. Всред бисери по злато  

1. Ще дойде ден когато  
Ще бъда аз блажен;  
Всред бисери по злато  
Ще ходя променен. (3)  

2. Не ще смущават вече  
Душата ми злини;  
О, те ще са далече  
Христос ги отстрани. (3)  

3. Там ще пребъдвам вечно  
Всред прелести безброй;  
Доволен безконечно  
Потънал в мир, покой. (3)
