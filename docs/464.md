# 464. Както еленът извори  

*С. П. 405*  

1. Както еленът извори  
    На живата вода,  
Така желае, Боже, Теб  
    Примрялата душа.  

2. Пред Тебе, Боже в Твоя храм  
    Кога ще се явя,  
За да Те славословя там  
    С безкрайна веселба?  

3. Въздишам за честития  
    Преминал весел ден,  
Когато пред лицето Ти,  
    О, Боже, бях блажен.  

4. Душо, отпаднала ли си?  
    На Бога се надей:  
Той е спасението ти,  
    Ти Нему с радост пей.
